web-screenshot

一个浏览器扩展，可以选择页面中某个 DOM 元素进行截图，也可以框选某个区域进行截图

浏览器支持度

支持所有常见的基于 Chromium 的现代浏览器： Chrome、Edge、360急速浏览器、搜狗、QQ浏览器...

安装

点击打开软件主页 - Chrome 应用商店

推荐使用 Chrome 应用商店安装，如果你无法使用 Chrome 商店，则使用下面手动安装。

1. 点击下载程序安装包
2. 打开扩展程序管理页面
3. 将下载的程序直接拖拽到扩展程序管理页面中（浏览器会自动安装）
4. 将插件固定到顶部，方便不使用快捷键时点击图标进行截图
5. 重启浏览器

使用方法

点击扩展图标后可以选择截图方式或者对软件进行配置

也可以直接使用快捷键
1. Alt+Shift+D 选择DOM截图
2. Alt+Shift+Q 任意区域截图
